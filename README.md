## 简介  
ThinkPHP 是一个轻量级 PHP 开发框架。
**学习之前需掌握 html, php 及 mysql 相关知识，并有一定实践基础。**

ThinkPHP 目前有版本 3.2，不同版本文件结构相差较大。建议先学习 3.1 版本。遇到问题搜索时，注意版本号。
